Latihan Git

Ini adalah latihan dasar penggunaan Git.

Langkah-langkah:

  1. Inisialisasi repositori Git dengan perintah git init.
  2. Buat file baru bernama index.html.
  3. Tambahkan file ke staging area dengan perintah git add index.html.
  4. Lakukan commit dengan pesan yang sesuai menggunakan perintah git commit -m "Menambahkan index.html".
  5. Buat branch baru dengan perintah git branch nama-branch.
  6. Pindah ke branch tersebut dengan perintah git checkout nama-branch.
  7. Lakukan perubahan pada file dan ulangi proses add dan commit.
  8. Kembali ke branch utama dengan perintah git checkout main.
  9. Lakukan merge branch yang telah dibuat sebelumnya.

Perintah Git Dasar:

  • git init - Inisialisasi repositori Git baru.
  • git add - Menambahkan file ke staging area.
  • git commit - Menyimpan perubahan ke repositori.
  • git branch - Mengelola branch dalam repositori.
  • git checkout - Berpindah antar branch.
  • git merge - Menggabungkan perubahan dari branch lain.
  • git status - Menampilkan status repositori saat ini.
